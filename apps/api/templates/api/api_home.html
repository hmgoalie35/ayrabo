Getting Started

Authentication

All endpoints require authentication. This can be achieved through email/password or an api token.

Login and choose an api version below to start exploring the api.

You can also obtain an api token by POSTing your email and password to {{ obtain_api_token_path }}

Example request (your username is the same as your e-mail)

curl -X POST {{ obtain_api_token_path }} -d '{"username": "email-address","password": "password"}' -H "Content-Type: application/json"

Example response

{"token": "myapitoken"}

You can then use myapitoken as follows

curl -X GET /api/v1/endpoint/ -H 'Authorization: Token myapitoken'

Your api token will also be available on your account information page.

All requests against the api MUST have a trailing slash
